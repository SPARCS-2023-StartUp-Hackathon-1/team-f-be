## 💻 Build & Run

1. 빌드

```
$ ./gradlew build
```

2. 빌드된 파일 (*.jar) 실행

```
$ fairer-0.0.1-SNAPSHOT.jar
$ java -jar fairer-0.0.1-SNAPSHOT.jar
```

<br/>

## 📢 Commit message

`feat` : 새로운 기능에 대한 커밋

`fix` : 버그 수정에 대한 커밋

`refactor` : 새로운 기능 추가나 수정 없이 기존 코드 리팩토링

`docs` : 문서화에 대한 커밋
